# Dylan's Todos

This is a classic todo list project completed for the Design Engineer Engineering Track tutorial. 

## Tech
Written in HTML, Typescript, JavaScript, and React. Built in the Cursor code editor with minimal AI help. Hosted on Vercel.

## How to preview this project

### Locally on your machine:
1. Clone the repository on your local machine.
2. Enter terminal and install packages with: `npm i`
3. Run the local development server: `npm dev`

### Live on the internet:
1. Visit this URL: https://todo-app-dylanpmcadam.vercel.app/
